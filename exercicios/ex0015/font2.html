Como usar fontes externas baixadas

Para usar fontes baixadas externamente é preciso depois de baixa(s) usar um seletor especial chamado de @font-face, depois disso é necessário indicar com a propriedade font-family o nome que você vai dar para a fonte baixada.
Já no src é só indica onde estar a fonte baixada.

Da para perceber que lá no src há uma propriedade chamada de format, é nela que será especificada o formanto da fonte baixada.

Formados de fontes

Há varios formatos de fontes alguns deles são:

  • opentype(otf)
  • truetype(ttf)
  • embedded-opentype
  • truetype-aat (Apple Advanced Typography)
  • svg
